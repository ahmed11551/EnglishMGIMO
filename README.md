# МГИМО ENGLISH

Mini App для изучения профессиональной лексики по международным отношениям (МГИМО). Запускается как **Telegram Mini App** и в браузере.

## Возможности

- **10 тематических модулей** (230+ карточек):
  - Дипломатия и протокол, ООН и организации, Безопасность и кризисы, Основы МО, Государство и власть, Экономика и торговля, Международное право, Общество и культура, Академия, Фразы и общение

- **Режимы обучения:**
  - **Флэш-карты** — пролистывание карточек (термин ↔ перевод + пример), кнопка озвучки (TTS/audioUrl)
  - **Тренажёр** — выбор правильного перевода из 4 вариантов с учётом интервальных повторений
  - **Конструктор** — сборка термина из букв по переводу
  - **Аудирование** — прослушать термин (без текста) и выбрать перевод из 4 вариантов
  - **Произношение** — произнести термин вслух; проверка через Web Speech API (Chrome/Safari)

- **Интервальные повторения (SM-2)** — слова на повтор планируются по алгоритму запоминания.

- **Прогресс и статистика** — дневная цель (10 слов), серия дней, календарь недели, уровень/XP, выучено по модулям.

- **Telegram** — при запуске из Telegram прогресс привязан к `Telegram ID` (данные в `localStorage`). Бот с кнопкой «Открыть приложение» и опциональным вебхуком на /start.

## Запуск

```bash
npm install
npm run dev
```

Откройте в браузере указанный адрес (например `http://localhost:5173`).

Для проверки в Telegram: разверните приложение по HTTPS и укажите URL в [@BotFather](https://t.me/BotFather) как Web App для бота.

## Сборка

```bash
npm run build
```

Статика в каталоге `dist/`. **Деплой на Vercel и настройка бота:** пошаговая инструкция в **[docs/VERCEL_AND_BOT.md](docs/VERCEL_AND_BOT.md)**. Альтернативные хостинги — [docs/DEPLOYMENT.md](docs/DEPLOYMENT.md).

## Технологии

- **React 19** + **TypeScript** + **Vite**
- **React Router** для маршрутов
- **Telegram Web App API** (скрипт в `index.html`) — без отдельного npm-пакета
- Локальное хранение прогресса в **localStorage** (в будущем — бэкенд для синхронизации)
- **PWA** (vite-plugin-pwa): офлайн-доступ, кеширование статики и контента

## Расширение контента

- Карточки и модули задаются в `src/data/cards.ts` и `src/data/modules.ts`.
- Для добавления аудио: поле `audioUrl` в типе `Card` и проигрывание в компонентах.
- Режимы «Аудирование» и «Произношение» реализованы; для произношения нужен браузер с поддержкой Web Speech API (Chrome, Safari).
